Dialog with radios

Keyboard instructions
  • When a dialog opens, focus moves to an element inside the dialog.
  • Tab:
    • Moves focus to the next tabbable element inside the dialog.
    • If focus is on the last tabbable element inside the dialog, moves focus to the first tabbable element inside the dialog.
  • Shift + Tab:
    • Moves focus to the previous tabbable element inside the dialog.
    • If focus is on the first tabbable element inside the dialog, moves focus to the last tabbable element inside the dialog.
  • Escape: Closes the dialog.

Notes

  • When <input type="radio"> have the same name attribute, it is possible to navigate through each by Up Arrow, Down Arrow without adding event listeners.
  • role="dialog" hides everything outside the element for NVDA Links list.

Code

<button aria-haspopup="true">Menu button </button>
<div role="dialog" style="display:none" aria-label="Some dialog">
    <label>
        <input name="dialogRadio" type="radio">
        One
    </label>
    <label>
        <input name="dialogRadio" type="radio">
        Two
    </label>
    ...
    <button>Apply changes</button>
    <button>Close</button>
</div>
