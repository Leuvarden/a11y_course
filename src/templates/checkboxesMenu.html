Menu with checkboxes

Keyboard instructions

Menu:
  • Enter: Activates the item and closes the menu.
  • (Optional): When focus is on a menuitemcheckbox, changes the state without closing the menu.

Links

  • WAI-ARIA Menu or Menu bar
  • Menu Example

Code

<button aria-haspopup="true">Menu button </button>
<ul role="menu" style="display:none" aria-label="Some menu">
    <li role="menuitemcheckbox" tabindex="0">One</li>
    <li role="menuitemcheckbox" tabindex="0">Two</li>
    <li role="menuitemcheckbox" tabindex="0" aria-checked="true">Three</li>
    <li role="menuitemcheckbox" tabindex="0">Four</li>
</ul>
